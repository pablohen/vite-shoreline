# vite-shoreline

A Simpsons episode browser built with [Vite](https://vite.dev/), [React 19](https://react.dev/), and [VTEX Shoreline](https://github.com/vtex/shoreline). It lists episodes from [The Simpsons API](https://thesimpsonsapi.com) in a table with URL-backed pagination and a details drawer.

This project was developed using an agentic AI-assisted workflow (e.g. coding agents in [Cursor](https://cursor.com)).

## Features

- Paginated episode list; current page is stored in the query string (`?page=`)
- Episode details in a side drawer
- [TanStack Query](https://tanstack.com/query) for data fetching and caching
- In development, React Query and TanStack Router devtools are available (see the root layout)

## Prerequisites

- [Node.js](https://nodejs.org/) (LTS recommended)
- [pnpm](https://pnpm.io/) (this repo uses a `pnpm-lock.yaml`)

## Scripts

| Command | Description |
| --- | --- |
| `pnpm dev` | Start the Vite dev server with HMR |
| `pnpm build` | Typecheck and production build to `dist/` |
| `pnpm preview` | Serve the production build locally |
| `pnpm typecheck` | Run `tsc --noEmit` |
| `pnpm lint` | Run Biome check |
| `pnpm format` | Format with Biome |

## Stack

- **Build:** Vite, `@vitejs/plugin-react`, TanStack Router Vite plugin (file-based routes, generated `src/routeTree.gen.ts`)
- **UI:** React, Shoreline (`@vtex/shoreline`, `@vtex/shoreline-ts-table`)
- **Data:** TanStack React Query, API helpers in `src/simpsons-api.ts`
- **Routing:** TanStack React Router, Zod-validated search params (e.g. `page`) on the root route
- **Lint & format:** [Biome](https://biomejs.dev/) (`biome.json`)

## Project layout

- `src/routes/` — TanStack Router routes; search state and layout live in `__root.tsx`
- `src/components/` — Episodes table, page shell, and episode details drawer
- `src/simpsons-api.ts` — Types and fetch helpers for the public API
- `src/routeTree.gen.ts` — Generated by the router plugin (do not edit by hand)

Built with Vite and the React plugin.
