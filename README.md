# vite-shoreline

A Simpsons data browser built with [Vite](https://vite.dev/), [React 19](https://react.dev/), and [VTEX Shoreline](https://github.com/vtex/shoreline). It browses [The Simpsons API](https://thesimpsonsapi.com) in three **Episodes**, **Characters**, and **Locations** tabs, each with a paginated table, URL-backed search state, and a side details drawer.

This project was developed using an agentic AI-assisted workflow (e.g. coding agents in [Cursor](https://cursor.com)).

## Features

- Three tabs; active tab and page are stored in the query string (`?tab=`, `?page=`, with defaults `tab=episodes` and `page=1`)
- Paginated tables per resource (page size `SIMPSONS_PAGE_SIZE` in the API client)
- Row details in a side drawer (episode, character, or location)
- [TanStack Query](https://tanstack.com/query) for data fetching and caching; only the **active** tab’s list query runs (`enabled` per tab)
- In development, React Query and TanStack Router devtools are available (see the root layout)

## Prerequisites

- [Node.js](https://nodejs.org/) (LTS recommended)
- [pnpm](https://pnpm.io/) (this repo uses a `pnpm-lock.yaml`)

## Stack

- **Build:** Vite, `@vitejs/plugin-react`, TanStack Router Vite plugin (file-based routes, generated `src/routeTree.gen.ts`)
- **UI:** React, Shoreline (`@vtex/shoreline`, `@vtex/shoreline-ts-table`)
- **Data:** TanStack React Query, list and detail helpers in `src/simpsons-api.ts`
- **Routing:** TanStack React Router, Zod-validated search params (`page`, `tab`) on the root route
- **Lint & format:** [Biome](https://biomejs.dev/) (`biome.json`)

## Project layout

- `src/routes/__root.tsx` — Search state, root layout, devtools
- `src/pages/SimpsonsBrowsePage.tsx` — Tab shell, queries, drawer state
- `src/components/` — `*TabPanel`, `*DetailsDrawer`, and table UI
- `src/utils/` — Shared helpers (e.g. error and collection view helpers)
- `src/simpsons-api.ts` — Types and fetch helpers for the public API
- `src/simpsons-tabs.ts` — Tab IDs and Zod schema for the `tab` search param
- `src/routeTree.gen.ts` — Generated by the router plugin (do not edit by hand)
